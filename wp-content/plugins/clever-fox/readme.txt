== Description ==

Clever Fox plugin to enhance the functionality of WordPress themes made by Nayra Themes. 40000+ users are using this plugin for web design. Main motive behind this plugin is to boost up functionality of Nayra themes and focus on a smooth user experience. Clever Fox contains all features which are required to create a complete website. Clever Fox comes with everything you could ever need to build an amazing website that is suitable for business, portfolio, food & restaurant, gym & fitness, spa salon, medical practitioner & hospitals, WooCommerce Websites. You can see below listed free themes.

= The Clever Fox plugin features include: =

- Import Pre-made Section in 1 Click
- Beautiful Sections on Homepage
- Edit content via Live Customizer
- Manage Font Typography
- Background Animations
- Kenburn Effect on Slider
- Compatible with WPML & Polylang
- Translation Ready
- Responsive for All Devices
- Cross Browser Compatibility
- and More Features

== Installation ==

= Option 1. =

- Upload the entire `clever-fox` zip file to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Clever Fox" —> Click install.
- After plugin is downloaded, click active plugin
